2015-12-09
Web_Development

This site is created with Hexo, a static site generator and hosted with github page. Hexo is a simple blog framework that allows us to write post easily and generates static files quickly with beautiful theme. Github page allows us to host our static site for free. If you have been using Github as your source code repo, then you can setup this in a second. You can setup one using the information in both links included above.

Hexo’s blog post is posted using Markdown format. Markdown allows us to write in simple to read plain text format and convert to html format. Depending on the markdown rendering engine and the markdown flavour, the non-standard syntax could be different, You can see the original Markdown information here.
